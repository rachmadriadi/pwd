Aplikasi web sederhana untuk menghasilkan password yang kuat, mudah diingat, dan unik menggunakan nama-nama gunung di Indonesia.

``âœ¨ Fitur``

Basis Nama Gunung: Menggunakan database nama gunung populer di Indonesia (Semeru, Rinjani, Prau, dll).

Algoritma Leet Speak: Mengubah huruf menjadi angka/simbol secara acak (misal: A -> @, E -> 3).

Repetisi Cerdas: Mengulang nama gunung dengan variasi unik untuk memenuhi panjang karakter tanpa karakter acak yang membingungkan.

Support PostgreSQL: Bisa menghasilkan output query CREATE USER, ALTER USER, atau format .pgpass untuk database administrator.

Tampilan Modern: Desain Glassmorphism yang responsif (bisa dibuka di HP).
